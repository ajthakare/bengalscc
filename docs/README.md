# Documentation

This folder contains all project documentation, organized into the following structure:

## Folders

### `/admin`
Admin panel setup and usage documentation:
- `ADMIN_SETUP.md` - Complete guide for setting up and using the admin interface

### `/guides`
Setup and deployment guides:
- `QUICKSTART.md` - Quick start guide for developers
- `CONFIG_GUIDE.md` - Configuration guide
- `DEPLOYMENT_CHECKLIST.md` - Pre-deployment checklist and steps

### `/design`
Design specifications and content:
- `Bengals_CC_Website_Wireframes.docx` - Original wireframes (Word)
- `Bengals_CC_Website_Wireframes.txt` - Wireframes (text format)
- `Bengals_Cricket_Club_Website_Content.txt` - Website content specifications

### `/assets`
Documentation assets:
- `logo.svg` - Club logo (reference copy - source is in `src/assets/logo.svg`)
- `fixtures.csv` - Fixtures data format example
- `messages.png` - Screenshot for documentation

## Root Level Files

### Theme Documentation
- `THEME_IMPLEMENTATION.md` - Theme implementation details
- `THEME-GUIDE.md` - Theme customization guide
- `THEMING.md` - Theming overview

### Project Management
- `IMPLEMENTATION_SUMMARY.md` - Summary of implemented features
- `TODO.md` - Project todos and future enhancements
- `newideas.md` - Ideas and feature requests
